## 🚀 快速开始

### 1. 查看当前配置
配置文件位置：`c:\Users\swj\.cursor\mcp.json`

### 2. 已启用的工具
- ✅ **filesystem** - 文件系统增强操作
- ✅ **memory** - 持久化知识库
- ✅ **puppeteer** - 浏览器自动化（需Chrome）
- ✅ **git** - Git版本控制（新增）

### 3. 使用方式
直接与AI对话，AI会自动选择合适的MCP工具：

**示例**：
```
# 文件操作
"搜索项目中所有TODO"

# Git操作
"查看当前Git状态"
"提交所有更改，消息为'完成功能'"

# 浏览器操作
"访问百度首页并截图"

# 知识管理
"记住我们的项目使用FastAPI框架"
```

## ⚠️ 重要提示

### 修改配置后必须重启
每次修改`mcp.json`后，必须**完全关闭并重新打开Cursor**。

### Git工具刚刚配置
**需要重启Cursor**才能使用Git MCP工具！
